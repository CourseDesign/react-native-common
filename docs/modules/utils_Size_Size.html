DirectionalSizeUnit

DirectionalSizeUnit: `${SizeUnit} ${SizeUnit}` | `${SizeUnit} ${SizeUnit} ${SizeUnit} ${SizeUnit}`
  • Defined in src/utils/Size/Size.ts:61

방향성이 있는 크기를 나타내는 타입으로 여백을 통해 구분됩니다. 주로 padding, margin에 사용됩니다. 1방향, 2방향, 4방향을 나타낼 수 있습니다.

example
// 위로 30% 아래로 24px
const padding2: DirectionalSizeUnit = '30% 24px';
// 위로 30% 오른쪽으로 24px 아래로 2(기본값) 왼쪽으로 3unit
const padding3: DirectionalSizeUnit = '30% 24px 2 3unit';
